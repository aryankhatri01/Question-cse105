# Student Grade Filter

## Objective
Categorize student scores using an ArrayList and conditional statements.

## Concepts Used
- ArrayList
- For-each Loop
- If-Else Statements

## Logic
1. Store exam scores in an ArrayList.
2. Traverse the list using a for-each loop.
3. Classify scores:
    - 90 and above → Distinction
    - 40 to 89 → Pass
    - Below 40 → Fail

## Sample Output

95 : Distinction
82 : Pass
38 : Fail
91 : Distinction
65 : Pass
25 : Fail
